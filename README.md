# ProjetoAgrinho 🌱

Um site básico dedicado à educação e conscientização sobre agricultura sustentável.

## Características

- 🎨 **Design Responsivo**: Funciona perfeitamente em dispositivos móveis e desktop
- ⚡ **Leve e Rápido**: Apenas HTML, CSS e JavaScript vanilla
- 🌿 **Temática Agrícola**: Cores e emojis relacionados à natureza
- 📱 **Acessível**: Estrutura semântica e boa usabilidade

## Estrutura do Site

### Páginas/Seções

1. **Navegação** - Menu principal com links para diferentes seções
2. **Hero Section** - Boas-vindas e chamada para ação
3. **Sobre** - Informações sobre o projeto
4. **Serviços** - Apresentação dos principais serviços
5. **Contato** - Formulário para entrar em contato
6. **Rodapé** - Informações de copyright

## Arquivos do Projeto

- `index.html` - Estrutura HTML principal
- `style.css` - Estilos e design responsivo
- `script.js` - Funcionalidades interativas
- `README.md` - Documentação do projeto

## Como Usar

### 1. **Localmente**
```bash
# Clone o repositório
git clone https://github.com/nilmarmuller-coder/ProjetoAgrinho.git

# Abra o arquivo index.html no seu navegador
# Você pode simplesmente fazer duplo clique no arquivo
```

### 2. **GitHub Pages**
1. Acesse as configurações do repositório
2. Vá até "Pages"
3. Selecione a branch "main" como fonte
4. Seu site estará disponível em: `https://nilmarmuller-coder.github.io/ProjetoAgrinho/`

## Personalização

### Mudar Cores
Edite as variáveis CSS em `style.css`:
```css
:root {
    --primary-color: #2ecc71;      /* Verde principal */
    --secondary-color: #27ae60;    /* Verde secundário */
    --dark-color: #2c3e50;         /* Escuro */
    --light-color: #ecf0f1;        /* Claro */
}
```

### Adicionar Mais Serviços
No `index.html`, adicione mais cards na seção de serviços:
```html
<div class="service-card">
    <h3>🌾 Novo Serviço</h3>
    <p>Descrição do novo serviço</p>
</div>
```

### Modificar Conteúdo
Simplesmente abra `index.html` e edite o texto, títulos e descrições conforme necessário.
